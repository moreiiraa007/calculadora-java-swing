# 🧮 Calculadora Java Swing

Calculadora com interface gráfica desenvolvida em Java utilizando Swing.

## 🚀 Funcionalidades
- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 🧹 Limpar campos
- ⚠️ Tratamento de erros
- ❌ Validação de divisão por zero

## 🛠 Tecnologias
- Java
- Swing
- IntelliJ IDEA

## 🎯 Objetivo
Praticar desenvolvimento com interface gráfica em Java, eventos e organização de código.
